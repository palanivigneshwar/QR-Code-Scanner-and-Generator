# QR-Code-Scanner-and-Generator
The following app is used to generate and scan qr code
In text editor type the word and qr code is generated using zxing library
Generator.java-Displays the text editor from where the text will be converted to qr code
Main2Activity.java-Displays the encoded message
MainActivity.java-Main page connects the generator,scanner and credits to the front page
Qr.java-Displays the qr code
